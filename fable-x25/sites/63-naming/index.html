康熙筆畫 · Strokes

Traditional strokes康熙字典 counts

Every grid is built from a character's stroke count in its 康熙字典 traditional form — not its modern simplified shape. Radicals count by their root: 氵=水(4), 艹=艸(6), 辶=辵(7). A bundled table holds a few hundred common characters.

五格 · Five Grids

The five numbersthe formulas

天格 = surname (+1 if single); 人格 = last surname char + first given char; 地格 = given name (+1 if single); 總格 = every stroke summed; 外格 = the remainder — the outer influence. Rules shift for 1-, 2- and 3-character names.

數理 · 81 Numbers

The 81 fortunes吉 / 半吉 / 凶

Each grid number lands on one of the 八十一數理, each marked 吉 auspicious, 半吉 mixed, or 凶 inauspicious with a short meaning. Numbers above 81 fold back by subtracting 80.

三才 · Three Powers

Element & flow木火土金水

天·人·地 reduce to a Five Element by their last digit (1-2 木, 3-4 火, 5-6 土, 7-8 金, 9-0 水). The flow between them — generating 相生, controlling 相剋, or alike 比和 — sets the 三才配置.

A note on honesty. The 五格 formulas and 81-數理 table are standard and applied exactly; the stroke table is curated for common characters using published 康熙 values, and a rare or ambiguous character may differ by a stroke — where a character is unknown, the report flags it and lets you set its stroke count by hand, then recomputes. The 三才 verdict here is derived directly from 五行 生剋 relationships rather than a lookup table. Read it as a beautifully-made mirror, not a machine of fate.
